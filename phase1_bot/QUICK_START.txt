## 📋 What You Need to Add (2 things only)

### 1️⃣ MongoDB Connection
```
Get from: mongodb.com/cloud/atlas
Create: Free M0 cluster
Copy: Connection URI
Add to .env as MONGO_URI=
```

### 2️⃣ Your Crypto Addresses
```
ESCROW_BTC_ADDRESS=    (coinbase/kraken wallet)
ESCROW_USDT_ADDRESS=   (ethereum wallet)
ESCROW_LTC_ADDRESS=    (litecoin wallet)
```

**At least ONE required, all three recommended**

---

## 🚀 3-Minute Setup

```bash
cd phase1_bot
cp .env.example .env

# Edit .env:
# Add MONGO_URI (from MongoDB Atlas)
# Add your wallet addresses

# Run:
pip install -r requirements.txt
python bot/main.py
```

**Bot starts once required env vars are set. ✅**
